## About The Project

[![Product Name Screen Shot][product-screenshot]](https://github.com/imcousin/mmd_tech_challenge_nov_2021)


An application that allows students to submit assignments and the instructor can mark the assignments on the website.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [MongoDB](https://www.mongodb.com/)
* [Express.js](https://expressjs.com/)
* [Node.js](https://nextjs.org/)
* [React.js](https://reactjs.org/)
